Pizza API Documentation

Pizzas

GET /api/pizzas
Retrieves a list of all pizzas.

Responses

Status Code Description Schema
200 OK Successful operation Array[Pizza]
500 Internal Server Error Error fetching pizzas Error
POST /api/pizzas
Creates a new pizza.

Request Body

Parameter Type Description
name string Name of the pizza (required)

Responses

Status Code Description Schema
201 Created Pizza created successfully Pizza
400 Bad Request Invalid pizza name Error
500 Internal Server Error Error adding pizza Error
GET /api/pizzas/{id}
Retrieves a specific pizza by ID.

Parameters

Parameter Type Description In
id integer ID of the pizza to retrieve path

Responses

Status Code Description Schema
200 OK Successful operation PizzaDetails
500 Internal Server Error Error fetching pizza Error
DELETE /api/pizzas/{id}
Deletes a specific pizza by ID.

Parameters

Parameter Type Description In
id integer ID of the pizza to delete path

Responses

Status Code Description
200 OK Pizza deleted successfully
500 Internal Server Error Error deleting pizza

Pizza Toppings

POST /api/pizzas/{id}/toppings
Updates the toppings for a specific pizza.

Parameters

Parameter Type Description In
id integer ID of the pizza to update toppings for path

Request Body

Parameter Type Description
selectedToppings Array[string] List of toppings to add to the pizza

Responses

Status Code Description
200 OK Toppings updated successfully
400 Bad Request Invalid toppings format
500 Internal Server Error Error updating toppings

Schemas

Pizza

Property Type Description
id integer Unique identifier for the pizza
name string Name of the pizza

PizzaDetails

Property Type Description
pizzaName string Name of the pizza
selectedToppings Array[string] List of selected toppings for the pizza

Error

Property Type Description
error string Error message
